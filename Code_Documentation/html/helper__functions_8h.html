Remove all the vertices in the given polygon that are on the same side of the line formed by first and last vertices as the given inside index.

The Function modifies the vertex list by removing all the vertices that lie in the same semi plane as the given inside vertex The Semi Plane is the line formed by first and last vertices of the polygon and inside vertex must lie on the opposite side of the line as the remaining vertices
