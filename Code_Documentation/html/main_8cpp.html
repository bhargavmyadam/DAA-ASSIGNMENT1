The Function Splits the Polygon into Convex Polygon using Split Algorithm.

Parameters

The function starts with all set of vertices and selects consecutive vertices from the start of original vertices list and forms a convex polygon by adding diagonal and then takes the rest of the polygon excluding the vertices of the newly formed convex polygon and performs the same function on the rest of the vertices.
