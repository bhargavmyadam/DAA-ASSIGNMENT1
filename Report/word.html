ABSTRACT

Decomposing an arbitrary polygon into smaller convex polygons holds a lot of value in literature due to its applications in various domains such as computer graphics, geographical information systems, robotics etc. This project focuses on developing a model to achieve the above task by implementing an algorithm that doesn’t add additional vertices to the polygon. This divide and conquer approach-based algorithm adds required edges between vertices not connected to each other to remove the presence of internal reflex angles and partition the polygon into convex polygons. We also implement a merging heuristic algorithm that identifies some inessential edges added by the algorithm and removes them to generate a more optimal partition. We implemented the Doubly Connected Edge List (DCEL) data structure to store the polygon decomposition related information. We experimented the developed algorithm on instances of varying complexity and performed a timing analysis to observe the performance of the algorithm on such instances.

ALGORITHM IMPROVEMENT

Algorithm used in the paper:

According to the research paper in the merge step, where we remove unnecessary diagonals created during split step, it suggests removing a diagonal only if the vertices of the diagonal have at least 2 diagonals incident on them.

Improved algorithm from our findings:

We modified the algorithm slightly in such a way that we remove diagonals when the vertices of diagonal have at least one diagonal incident on them instead of 2 diagonals as suggested in the paper.

We tested both the versions of the algorithm on some test cases and observed better results in our algorithm.
